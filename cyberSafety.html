What is Phishing


Phishing is when scammers use text messages or emails to trick people into giving them personal information. With that information (passwords, account numbers, Social Security numbers), scammers can get into emails and bank accounts or commit identity theft. Thousands of phishing attacks are sent out everyday. One year people lost $57 million to phishing scams

Phishing emails and texts can look as if a known or trusted company sent it. Phishing messages usually tell a story to trick people into clicking a link or opening an attachment. Opening phishing links/attachments can download harmful malware. Phishing attacks can harm the reputation of the companies that scammers are pretending to be. Phishing Emails’ Common Attributes include: Too good to be true, urgent messages, hyperlinks, attachments, unusual senders.

How to Avoid Being Phished


Email spam filters already keep many phishing emails out of inboxes

Steps for Extra Protection:
1. Protect you computer with security software
2. Protect your phone by making software update automatically
3. Protect your accounts through multi-factor authentication (utilizes two or more confirmation to provide extra security)
4. Protect your data by backing it up


Verify if you have an account with the company that messaged you before opening links

If you don’t then it could be a phishing scam, but if you do make sure to verify that it’s real using the company’s confirmed phone number or website

Change passwords on a regular basis

Don’t use the same password for different accounts


Set browser settings to prevent fake websites from opening

Secure websites with valid SSL certificates begin with “https”






What to do if you responded to a phishing scam


-Go to IdentityTheft.gov if a scammer has your information

-If there is harmful malware on your computer, update your antivirus software to scan for viruses

-Report phishing messages to reportphishing@apwg.org and to ftc.gov/complaint or forward the text to SPAM (7726)









What is Spyware


Spyware is a type of software (malware) that gets installed on a device without the user knowing. It invades it, steals information and data & gives it to advertisers, data firms, or external users.

Any software is considered spyware if it is downloaded without the user’s authorization. Once it is installed, it monitors internet activity, tracks login information and spies. Often, the main goal is to obtain credit card numbers, banking info, and passwords. It's usually difficult to detect; first sign in a reduction in processor or network connection speeds. As for mobile devices, it slows down data usage and battery life.

How Spyware Works


Ways to get infected include, but are not limited to pirating media, games, videos, and music; downloading materials from unreliable or unknown sources; accepting a pop-up advertisement without reading the content; accepting and opening emails from unrecognized senders.

It's least damaging form exists as an application that runs as soon as the device is turned on and continues to run in the background. It steals information from RAM and processor power and may generate infinite pop ups, slowing down web browser until it become unusable. It may continually pop up ads making search engine impossible to use; can change a computer’s dynamically link libraries (DLL) (which results in connectivity failure that is hard to diagnose). In its most damaging form, spyware tracks web browsing, record works, password, sensitive information, leading to identity theft.





















How to Prevent it.


- Download software from trusted sources only
- Reading all disclosure when installing software
- Don’t interact with pop-up adds
- Staying current with updates and patches for browser, OS and application software
- Download anti spyware tools, use extensive and reputable antivirus software, avoid opening emails from unrecognized senders, and enable two-factor authentication (2FA) whenever possible
- Anti Spyware tools

Malwarebytes: can remove spyware; scans through registry files, running programs, hard drives, and individual files
Trend Micro HouseCall: does not require download, uses minimal processor and memory resources and disk space; scans computer
Windows Defender: antimalware Microsoft program in Windows 10; lightweight tool that protects against threats like spyware, adware and viruses









How to Remove Spyware


Symptoms

Device runs slower
Device crashes unexpectedly
Pop-up ads appear when user is both online and offline
Device begins to run out of hard drive space

What to do

Disconnect internet connection
Check device’s program lists to see if the unwanted software is listed; choose to remove it; then reboot
If above does not work, run a scan: scan will ask user to remove or clean software
Download a virus removal tool or anti spyware tool
If none work, user has to access device’s hard drive in safe mode -> Requires a tool that will enable the user to access spyware folders and manually delete them
